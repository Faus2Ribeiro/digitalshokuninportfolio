Veet's EasierWax Epilator

My first ever project using Autodesk Fusion 360. For the project, I, along with a partner, had to create a women's epilator following the principles of Good Design set by Dieter Rams.
Our end result was a user-friendly epilator, where the user slides in a wax slab through the back. Inside, there's a battery that melts the wax. We also added a gyroscope feature and sensors which makes the machine control the quantity of wax that comes out.
